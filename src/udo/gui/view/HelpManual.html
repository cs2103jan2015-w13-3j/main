Command Description
Add
To add a new task, type the “add” command followed by task’s content, the type of task, necessary time and date information for that type of task, label and priority. Each part of the information provided in the command text should be preceded by its corresponding option delimiter.
[add] [task’s content] [/start <date and time>] [/end <date and time>][/deadline <date and time>] [/duration <hours and minutes>] [/reminder <date and time>] [/important]
Delete
To delete a task, first “display” or “search” with the relevant information to display the tasks. Type “delete” command followed by the index of the task.
delete [task’s index]
Undo
Simply undo!
undo
Modify
Each relevant task will be displayed on the GUI preceded by an index. Type in the modify command with the following format:
modify [task’s index] [new task’s content] [other options which are supported in “add” command]
Mark As Done
Each relevant task will be displayed on the GUI preceded by an index. Type done followed by a valid index to mark a task as done.
done [task’s index]
Unconfirmed Tasks
Each relevant task will be displayed on the GUI preceded by an index. Type done followed by a valid index to mark a task as done.
[task's content][[/deadline][/start][/end] <date and time> or <date and time> or <date and time>...]
confirm [task’s index]
Search
To search for a task, type “search” followed by the followed by the search content. The program will attempt to infer what the user want to search from the content’s text The program will first separate the search content’s text into individual words. All tasks’ content that match any of the word will be included in the search results sorted by the percentage matched. For each word, the program will support wild-card matching and near- match. Supported wild-cards are ‘?’ which denotes any single lowercase or uppercase character or digit, ‘*’ which denotes any sequence of characters and digits that do not include space, tab or new line characters. Near-match searching is determined by the degree of differences between the searched word with the content’s word and also the word’s length. A list of dates will also be extracted from the search content. Any task that has the deadline, start or end date within period of 1 day before and after any one of the extracted date will also be considered as a match.
search [content]
Autocomplete
When a user is typing a command, the status line will automatically display suggested words and phrases to be autocompleted. The priority of the class of words (command names, keywords, options, english dictionary words) suggested will be inferred from the current text being entered.

Up/Down arrow key: Change the input text to the previous command and next command in command history

Tab key: Pressing Tab can result in 2 cases. If the current input text consists of only the modify command and an index, the existing task’s information with that index will be autocompleted. Otherwise, the first suggested word will be autocompleted in place of the last word in the input text.

modify [task's index] [/deadline][/start][/end][/reminder]*tab*
Reminder
Setting a reminder will alert the user at the specified timing
[/reminder <date and time>]
Hotkeys
F1 - Help
F2 - Display
